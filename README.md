# Build with AI - Website Builder Template
A minimal, professional website template designed for AI-assisted customization with Claude Code.
Show less
## What's Included
- **4 pages**: Homepage, Services, About, Contact
- **Responsive design**: Mobile, tablet, desktop
- **Components**: Hero, feature cards, package cards, testimonials, FAQ accordion, contact form, stats bar
- **No frameworks**: Pure HTML, CSS, JavaScript
- **Easy theming**: Change 6 CSS variables to rebrand
## Quick Start
1. Clone this repo
2. Open `CLAUDE.md` for customization instructions
3. Replace `{{PLACEHOLDER}}` values with your business info
4. Replace images in `images/` folder
5. Deploy to Netlify, Vercel, or GitHub Pages
## Customization
See `CLAUDE.md` for detailed instructions on:
- Changing brand colors
- Replacing business info placeholders
- Adding new pages
- Setting up form submissions
- Component reference guide
## Stack
- HTML5
- CSS3 (Custom Properties)
- JavaScript (ES6+)
- Google Fonts (Playfair Display + Montserrat)
